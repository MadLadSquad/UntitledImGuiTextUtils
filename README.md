A bunch of small utility functions to make marking down text in ImGui easier. We have the following rendering features:
- [x] Bold
- [x] Italic
- [x] Bold Italic
- [x] Monospace
- [x] Small
- [x] Link
- [x] Underline
- [x] Strikethrough
- [x] Blockquote
- [x] Code blocks
- [x] Inline code
- [x] Subscript
- [x] Superscript
- [x] Highlighted text
- [x] Ruby text
